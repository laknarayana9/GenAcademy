# Occupancy Eligibility Guidelines (Synthetic)

> SYNTHETIC DATA. Fictional guidelines for demonstration only.

## Section 2.1 — Acceptable Occupancy

Owner-primary and owner-secondary occupancy are eligible for HO3 without an
occupancy-based referral. Seasonal occupancy is eligible but may attract hazard
referrals depending on location and time vacant.

## Section 2.2 — Vacant Dwellings (RC-101)

Vacant dwellings are ineligible for the HO3 program. A submission with vacant
occupancy is a hard knockout and must be declined under reason code RC-101.
Vacant risks require a dwelling-fire or vacant-property program instead.

## Section 2.3 — Rental Occupancy (RC-102)

Rental (tenant-occupied) dwellings are not eligible for HO3, which is a
homeowner form. Rental occupancy must be referred to the landlord/DP3 product
under reason code RC-102. This is a referral rather than a knockout because an
alternative product is available.

## Section 2.4 — Occupancy Verification

If occupancy cannot be determined from the submission, intake must request it as
a required follow-up fact before the run can proceed. Missing occupancy is a
missing-information pause, not an eligibility decision.
