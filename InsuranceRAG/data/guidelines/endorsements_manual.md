# Endorsements Manual (Synthetic)
**Carrier:** DemoCarrier  
**Product:** HO3 / HO5  
**State:** CA (demo)  
**Effective Date:** 2026-01-01  
**Version:** v0.1-synthetic

## 1. General Rules
- Endorsements **MUST** be allowed for the product/state/effective date.
- Any endorsement labeled “UW Approval Required” **SHALL** trigger referral.

## 2. Water & Flood Related
### 2.1 Water Backup Coverage (WBK-01)
**Type:** Coverage Add  
**Eligibility:**  
- Property **MUST NOT** have ≥ 2 water losses in 36 months without mitigation.  
- If basement present, WBK-01 **SHOULD** be recommended.

**Requirement:**  
- If non-weather water damage claim exists in past 36 months, WBK-01 **SHALL** be referred for UW decision.

### 2.2 Flood Exclusion Acknowledgement (FLX-ACK)
**Type:** Condition / Disclosure  
- If the risk is in SFHA, FLX-ACK **MUST** be attached.  
- SFHA is determined via external flood zone tool or authoritative mapping.

## 3. Wildfire / Brush
### 3.1 Wildfire Deductible Endorsement (WFD-10)
**Type:** Deductible Change  
- If wildfire score **High**, WFD-10 **MAY** be applied.  
- If wildfire score **Severe** and UW approves, WFD-10 **MUST** be applied with defensible space documentation.

## 4. Ordinance or Law (ORD-01)
**Type:** Coverage Add  
- ORD-01 **SHOULD** be recommended for homes built before 2000.  
- If construction year is missing, agent **MUST** request it in the Missing-Info Loop before recommending.

## 5. Home Business (HOB-02)
**Type:** Coverage Add  
- Home business endorsement **SHALL** require UW approval if customer foot traffic is present.  
- Otherwise, HOB-02 **MAY** be offered.

## 6. Scheduled Personal Property (SPP-01)
**Type:** Coverage Add  
- If any single item value > $10,000, SPP-01 **SHOULD** be recommended and item list **MUST** be collected.
